# Your desk

This is a free workspace — not a lesson, nothing graded.

- **Code Studio** — an editor with your files on the left.
- **Terminal** — a real PowerShell 7 prompt (`PS /workspace>`).
- **Sage** — the guide at the next desk. Ask anytime; Sage can see what's on your screen.
- **git** — this folder is already a git repo, so you can commit as you go.

Do whatever you like: write a script, try some commands, sketch an idea. Everything
you do happens inside this desktop and nothing outside it is touched.
